# one-ring

A toy project, building a truly async file IO using io_uring, and on it building a custom event loop, as well as integration into asyncio's native event loop

## Overview

Welcome to the **one-ring** documentation.

This project is organised as a monorepo with the following packages:

- [**one-ring-core**](one-ring-core/overview.md) — `one_ring_core`
- [**one-ring-loop**](one-ring-loop/overview.md) — `one_ring_loop`
- [**one-ring-asyncio**](one-ring-asyncio/overview.md) — `one_ring_asyncio`
- [**one-ring-http**](one-ring-http/overview.md) — `one_ring_http`
